# Gramedia.id Collaboration Section

## Business Analysis & UI/UX Design Case Study

## Project Overview
This repository showcases the Business Analysis and UI/UX Design process for the **Collaboration Section** on **Gramedia.id**.

The project covers three business collaboration modules:

- B2B Gramedia
- Gramedia Reseller Program
- Gramedia Partnership

The objective was to translate business requirements into an intuitive and user-friendly interface while supporting Gramedia's digital business initiatives. The approved Adobe XD designs served as the primary reference for the external development vendor and were successfully implemented as part of the live Gramedia.id platform.
